# Dashboard

The dashboard is the landing page. It shows a grid of report cards summarizing the data currently in the warehouse.

## Cards

Each card renders a curated report — for example **AMR Resistance Rate** (percent resistant by antibiotic) and the **antibiogram** summary. Click a card to open the full report with a parameter bar and CSV export.

## Themes

Use the sun/moon toggle in the top bar to switch between dark and light themes. Your choice is remembered in the browser.
