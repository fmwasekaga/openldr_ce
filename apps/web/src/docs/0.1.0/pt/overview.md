# OpenLDR Community Edition

OpenLDR CE é um repositório de dados laboratoriais de código aberto dedicado à vigilância da resistência antimicrobiana (AMR). Ele ingere resultados laboratoriais em múltiplos formatos, armazena-os como FHIR R4 e produz relatórios alinhados ao WHO GLASS e exportações para o DHIS2.

## O que você pode fazer

- **Ingerir** arquivos SQLite do WHONET, HL7 v2 (ORU/ORM) e CSV/Excel por meio de plugins WASM em sandbox.
- **Gerar relatórios** sobre resistência AMR, antibiogramas, volume de testes, tempo de retorno e dados demográficos de pacientes — no painel ou exportados como CSV/PDF.
- **Enviar** dados agregados e de rastreamento para o DHIS2 e exportar arquivos WHO GLASS RIS.
- **Configurar** um depósito externo SQL Server ou Postgres como destino de relatórios.

## Por onde começar

Novo aqui? Leia **Primeiros Passos** para instalação e sua primeira ingestão. Configurando uma integração? Consulte **Relatórios Agregados DHIS2** e **Banco de Dados Externo**.
